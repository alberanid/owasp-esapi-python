get_named_entity(self, pbs)

source code 

Returns the decoded version of the character starting at index, or None if no decoding is possible.

Formats all are legal both with and without semi-colon, upper/lower case:

&aa; &aaa; &aaaa; &aaaaa; &aaaaaa; &aaaaaaa;

Parameters:
  • pbs - a PushBackString containing a named entity like &quot;
Returns:
the decoded version of the character starting at index, or None if no decoding is possible.
